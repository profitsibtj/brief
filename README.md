# Weekly Giveaway Brief Generator

Website sederhana untuk membuat brief design campaign weekly giveaway UniPin per region MY, PH, dan TH.

## Fitur

- Generate brief Instagram Feed 1080 x 1350.
- Input asset dan wording per region.
- Copy brief untuk tim design.
- Download brief `.txt`.
- Download CSV database.
- Save otomatis ke Excel SharePoint melalui Power Automate webhook.

## Cara Pakai

1. Buka website GitHub Pages.
2. Isi `Month / Theme`, `Game`, `Logo`, dan `Footer`.
3. Isi asset dan wording untuk MY, PH, dan TH.
4. Klik `Copy Brief` untuk kirim ke tim design.
5. Klik `Save to Excel` jika Power Automate URL sudah dipasang.
